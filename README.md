# 工科クエスト

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
* 必要なものがあれば追記してください（非推奨）

## ゲームの概要
* 勇者になって敵を倒せ！

## ゲームの遊び方
* コマンドを決定して出てくる敵を倒す。
* 勇者のHPが0になったらGAMEOVERになる。

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画
* 戦闘システム

* フィールドマップ移動時のランダムエンカウント機能。

* キャンパスマップ特定座標到達時のボス戦イベント発生機能。

* スペースキー押下による攻撃処理と、敵HPの管理、勝利判定。

### 分担追加機能
* 最初の拠点作成（担当：辻）：初の拠点の図を挿入。道に沿うように作成
* 森、ラスボスマップ作製（担当：かける）：
* 戦闘機能（担当：江澤、あさひ）：敵と戦う機能。こうげき、まほうなどを使用
